Welcome to FPL Dashboard's documentation!
=========================================

Overview
--------

The **FPL Dashboard** is a Python-based web application for visualizing Fantasy Premier League data. It provides detailed insights into player performances, costs, and fixtures using Dash.

Features
--------

- **Player Performance Analysis**: Compare player statistics across gameweeks.
- **Player Cost vs. Performance**: Understand the relationship between player cost and their performance.
- **ICT Index Breakdown**: Compare player Influence, Creativity, and Threat metrics.
- **Fixtures Difficulty Ratings**: Visualize and plan based on team fixture difficulties.
